# AI 新聞分類方案

## 分類類別

使用免費 AI（Hugging Face Inference API）自動分類新聞，包含以下 10 個類別：

1. **event-update** (事件更新)
   - 關於火災進展、救援情況、現場狀況、火勢控制等即時事件資訊

2. **financial-support** (經濟支援)
   - 資助、補助、津貼、賠償、基金、捐款、財政援助、現金援助等

3. **emotional-support** (情緒支援)
   - 心理輔導、社工服務、情緒支援熱線、創傷治療、哀傷輔導等

4. **accommodation** (住宿支援)
   - 庇護中心、臨時住宿、過渡性房屋、休息站、社區會堂等

5. **medical-legal** (醫療/法律支援)
   - 醫療服務、法律諮詢、法律援助、醫療站、義診等

6. **reconstruction** (重建資訊)
   - 重建進度、重建時間表、重建資源、重建計劃、恢復工作等

7. **statistics** (統計數據)
   - 死傷人數、失蹤人數、受傷人數、事件統計、數據更新等

8. **community-support** (社區支援)
   - 義工服務、物資收集、社區活動、民間組織支援、志願服務等

9. **government-announcement** (政府公告)
   - 政府部門發布的正式公告、政策、措施、安排等

10. **general-news** (一般新聞)
    - 其他與事件相關但無法歸類到上述類別的新聞

## 實現方式

- **AI 服務**：Hugging Face Inference API（免費，無需 API Key）
- **模型**：`facebook/bart-large-mnli`（零樣本分類模型）
- **備用方案**：如果 AI 分類失敗，使用關鍵詞匹配進行簡單分類

## 優勢

1. **自動化**：無需手動標記，AI 自動分類
2. **準確性**：使用先進的 NLP 模型，分類更準確
3. **免費**：Hugging Face Inference API 完全免費
4. **可擴展**：可以輕鬆添加新的分類類別

## 注意事項

- AI 分類可能需要幾秒鐘時間，如果超時會使用備用分類
- 首次調用模型時可能需要等待模型加載（冷啟動）
- 如果 Hugging Face API 不可用，會自動降級到關鍵詞匹配分類
